In acest document sunt prezentate tehnologiile si modul de lucru utilizare pentru realizarea proiectului

1.Frontend

Frontendul aplicatiei nostre este realizat in intregime cu ajutor tehnologiilor HTML5 si CSS3

1.1 HTML5-up

Html5-up este o comunitate care pune la dispozitia publicului, in maniera open-source o serie de templete-uri responsive frumos structurate. Template-ul preluat in cazul site-ului Bomberbot este Minimaxing.

Ilustrater

Majoritatea imaginilor care se gasesc pe pagina sunt realizate in programul Ilustrater, un concurent al Photoshop-ului.

2. Managementul codului prin Github / GitKraken

Conform cerintelor disciplinei, s-a folosit o platforma Git pentru a pastra si modulariza codul

GitHub este un Git web sau un serviciu de depozitare si control al versiunilor Internet a aplicatiilor aflate in development. Acesta oferă toate functionalitatile de control al codului sursa al Git-ului avand totusi si propriile caracteristici. Asigura controlul accesului la cod si numeroase trasaturi care favorizeaza colaborarea programatorilor cum ar fi: bug tracking, task management si wikiuri pentru proiecte.

GitKraken

GitKraken este cel mai popular Git Gui pentru Windows, Mac si Linux. "It made it easy for us to Unleash our repository".

3. Baza de date

Oracle 11g / 12c

Oracle este un sistem de gestiune a bazelor de date relationale bazata pe obiecte. Sistem produs si marcat de Corporatia Oracle.

SQL Developer

Aplicatia folosita pentru manipularea CRUD a bazei de date este tot cea furnizata de Oracle, anume Sql Developer in care se programeaza folosind limbajul PL/SQL.

4. PHP 7

Limbajul in care s-a programat partea de backend a aplicatiei este PHP. Hypertext Preprocessoreste un limbaj de scripting server-side folosit si dezvoltat in principal pentru aplicatii web, dar este folosit si pentru programarea tip general-purpose.

4.1. Apache 2.4

Interpretarea codurilor sursa este facuta de serverul Apache, versiunea 2.4.

5. Javascript

Jocul Bomberbot efectiv este realizat in Javascript. JavaScript (JS) este un limbaj de programare orientat obiect bazat pe conceptul prototipurilor. Este folosit mai ales pentru introducerea unor funcționalități în paginile web, codul Javascript din aceste pagini fiind rulat de către browser.,. Limbajul este binecunoscut pentru folosirea sa în construirea siturilor web, dar este folosit și pentru acesul la obiecte încastrate (embedded objects) în alte aplicații. Framework-ul folosit este Phaser, versiunea 2.6.2.
